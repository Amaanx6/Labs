**Experiment – 03**
**Configuring Static Routing Using GUI (Packet Tracer)**

**Aim**
To configure static routing using GUI in Cisco Packet Tracer and verify connectivity between different LAN networks.

**Requirements**

* Cisco Packet Tracer
* 3 Routers (R1, R2, R3)
* 3 Switches
* 6 PCs
* Copper straight-through and Serial DCE cables

**Procedure**

**1. Design Topology**

* Open Cisco Packet Tracer.
* Add 6 PCs, 3 switches, and 3 routers (R1, R2, R3).
* Connect PCs to switches using copper straight-through cables.
* Connect switches to routers.
* Connect routers using serial DCE cables.

**2. Configure Router IP (GUI)**

**R1**

* FastEthernet0/0: 192.168.1.100 / 255.255.255.0
* Serial0/0: 10.0.0.1 / 255.255.255.252

**R2**

* Serial0/0: 10.0.0.2 / 255.255.255.252
* FastEthernet0/0: 192.168.2.100 / 255.255.255.0
* Serial0/1: 11.0.0.1 / 255.255.255.252

**R3**

* Serial0/0: 11.0.0.2 / 255.255.255.252
* FastEthernet0/0: 192.168.3.100 / 255.255.255.0

**3. Configure PC IP Addresses**

**LAN 1**

* 192.168.1.1 – 192.168.1.4
* Subnet: 255.255.255.0
* Gateway: 192.168.1.100

**LAN 2**

* 192.168.2.1 – 192.168.2.2
* Subnet: 255.255.255.0
* Gateway: 192.168.2.100

**LAN 3**

* 192.168.3.1 – 192.168.3.2
* Subnet: 255.255.255.0
* Gateway: 192.168.3.100

**4. Configure Static Routes (GUI)**
Router → Config → Routing → Static.

Example routes:

**R1**

* Network: 192.168.2.0  Mask: 255.255.255.0  Next Hop: 10.0.0.2
* Network: 192.168.3.0  Mask: 255.255.255.0  Next Hop: 10.0.0.2

**R2**

* Network: 192.168.1.0  Mask: 255.255.255.0  Next Hop: 10.0.0.1
* Network: 192.168.3.0  Mask: 255.255.255.0  Next Hop: 11.0.0.2

**R3**

* Network: 192.168.1.0  Mask: 255.255.255.0  Next Hop: 11.0.0.1
* Network: 192.168.2.0  Mask: 255.255.255.0  Next Hop: 11.0.0.1

**5. Verification**

* Open Command Prompt in any PC.
* Use `ping` to test connectivity between networks.
* Use `tracert` to check the routing path.

Example:
`ping 192.168.3.1`
`tracert 192.168.3.1`

**Result**
Static routing configured successfully. Communication verified between different LAN networks using ping and tracert.
