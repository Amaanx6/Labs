**1. What is Breadth First Search?**
It explores a graph level by level, starting from the source node.

**2. How does BFS differ from DFS?**
BFS explores breadth-wise using a queue, DFS goes depth-wise using stack/recursion.

**3. What data structure is used in BFS?**
Queue.

**4. What is the time and space complexity of BFS?**
Time: O(V+E), Space: O(V).

**5. Can BFS be used to find the shortest path in weighted graphs?**
Only in unweighted or equally weighted graphs.

**6. How does DFS work?**
It explores deeper into a branch until no further nodes, then backtracks.

**7. What are the differences between BFS and DFS?**
BFS finds shortest path and uses queue; DFS uses stack and explores depth-first.

**8. What data structure is used to implement DFS?**
Stack or recursion.

**9. How can DFS be implemented recursively?**
Call the function for each unvisited adjacent node.

**10. Is DFS guaranteed to find the shortest path?**
No.
