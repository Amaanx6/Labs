Basic MongoDB Commands
1. Show Databases
show dbs


Output:

admin   40.00 KiB
config  72.00 KiB
local   72.00 KiB

2. Create a New Database
use MYDB1


Output:

switched to db MYDB1

3. Create a Collection
db.createCollection("students")


Output:

{ ok: 1 }

4. Insert a Document
db.students.insertOne({ "rollno": 501, "name": "cse1" })


Output:

{
  acknowledged: true,
  insertedId: ObjectId('66f255ec808c7f3e6bc73bf8')
}

5. Show Collections
show collections


Output:

students

6. Display Data
db.students.find().pretty()


Output:

[
  {
    "_id": ObjectId("66f255ec808c7f3e6bc73bf8"),
    "rollno": 501,
    "name": "cse1"
  }
]

7. Insert Another Document
db.students.insertOne({ "rollno": 502, "name": "cse2" })

8. Update a Document
db.students.updateOne({ rollno: 502 }, { $set: { name: "cse3" } })


Output:

{ acknowledged: true, matchedCount: 1, modifiedCount: 1 }

9. Delete a Document
db.students.deleteOne({ rollno: 111 })


Output:

{ acknowledged: true, deletedCount: 1 }

10. Drop Collection
db.students.drop()


Output:

true

11. Drop Database
db.dropDatabase()


Output:

{ ok: 1, dropped: 'MYDB1' }

Inserting Documents from JavaScript File

Create a file named db1.js with the following:

db.students.insertOne({ name: "Karthik", rollno: 101, marks: 98 });
db.students.insertOne({ name: "Ravi", rollno: 102, marks: 99 });
db.students.insertOne({ name: "Shiva", rollno: 103, marks: 100 });
db.students.insertOne({ name: "Pavan", rollno: 104, marks: 80 });


Then load it into MongoDB:

load("d:\\db1.js")


Output:

true


Display Documents:

db.students.find().pretty()
